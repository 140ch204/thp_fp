## Tester l'application 

  **En local :**
 
  > git clone https://github.com/140ch204/thp_fp

  > bundle install

  > git db:drop

  > git db:create 

  > git db:migrate

  > git db:seed

  > rails s


  Puis aller dans le navigateur à

  [http://localhost:3000/](http://localhost:3000/)
